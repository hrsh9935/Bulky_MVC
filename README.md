# Bulky App

Bulky App is a comprehensive book management and purchasing application developed using **ASP.NET Core** and **MVC architecture**. The application is designed to handle multiple user roles, including **admin**, **company**, **customer**, and **employee**, each with distinct access levels and functionalities. The app also integrates the **Stripe payment gateway** to facilitate secure transactions, ensuring a smooth and secure purchasing experience for customers and companies.

## Features

- **User Roles & Access Levels**:
  - Admin: Full access to manage users, products, and orders.
  - Company: Ability to create and manage their products and track orders.
  - Customer: Browse products, make purchases, and track their orders.
  - Employee: Assist with order processing and other backend tasks.
  
- **Stripe Payment Gateway Integration**:
  - Secure transactions with **immediate payments** and **refunds** for customers.
  - **Deferred payments** for companies, enhancing financial flexibility.
  
- **Comprehensive Book Management**:
  - Efficient management of books and their details (e.g., title, author, price, description).
  - Categorization and search functionality for easy navigation.
